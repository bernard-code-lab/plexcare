# plexcare-frontend

**Responsabilidade:** interfaces e aplicações cliente da PlexCare — web app, landing page e portais (médico, paciente, jurídico).

## Escopo
- Componentes de UI e design system (teal/dourado, dark-luxury)
- Páginas e fluxos de produto
- Integração com a API via `plexcare-backend`
- Acessibilidade (WCAG AA), performance e responsividade

## Stack prevista
- Vite + React + TypeScript
- Tailwind CSS, componentes no padrão shadcn (`@/components/ui`)
- Framer Motion

## Não é responsabilidade daqui
Regras de negócio, persistência, infraestrutura e observabilidade.
